# 建立個人課程倉儲
登入 github 後，在 taems 中利用老師給的連結建立名稱為 cd2024 的個人倉儲，完成後在設定內進入 Pages 設定改為 main 並儲存，讓github生成網站。

# Replit
用github登入replit，點擊左上 inport form github ，選擇自己的帳號及要加入的倉儲。


# 雙重認證
登入 github 在設定內進入 Password and authentication，綁定個人手機。
